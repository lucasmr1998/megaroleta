Como funciona
Via Sala (manual)
1. CEO abre a Sala e escolhe um agente
2. Digita em linguagem natural: “Crie um plano de lançamento para Timon”
3. Agente recebe: prompt + documentos + projetos + métricas
4. OpenAI gera resposta com blocos de ação
5. Sistema executa: cria projeto, etapas, tarefas, documentos
6. Tudo logado e visível no dashboard
Via Automação (cron)
1. Trigger dispara no intervalo configurado
2. Chama o agente designado (ex: Operador)
3. Agente executa a ferramenta (ex: Gerar FAQ)
4. Ferramenta coleta dados reais e processa
5. Resultado salvo + log registrado
6. Mesma ferramenta pode ser chamada na Sala manualmente
