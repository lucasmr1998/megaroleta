# Catalogo de Tools — Clube Megalink

> Ferramentas reutilizaveis que os agentes chamam durante seu trabalho.
> Cada tool faz UMA coisa e retorna resultado estruturado.

## Como Usar

1. Abra conversa com o agente desejado
2. Quando o agente precisar de uma tool, cole o prompt da tool na conversa
3. Passe o input que a tool pede
4. A tool retorna resultado estruturado que o agente incorpora no readout

## Catalogo

| Tool | Arquivo | Quem Usa | O que Faz |
|------|---------|----------|-----------|
| **Analise de Dados** | `analise_dados.md` | Todos | Consultar metricas do sistema e gerar insights |
| **Gerador de Copy** | `gerador_copy.md` | CMO, CS, B2B | Gerar copy para WhatsApp, redes sociais, materiais |
| **Calculadora ROI** | `calculadora_roi.md` | CFO, CPO, CMO | Calcular ROI, payback e cenarios financeiros |
| **Auditor de Codigo** | `auditor_codigo.md` | CTO | Revisar codigo com checklist de seguranca e performance |
| **Gerador de Spec** | `gerador_spec.md` | CPO, CTO | Gerar spec completa de feature com criterios |
| **Prospector de Parceiro** | `prospector_parceiro.md` | Comercial B2B | Gerar abordagem personalizada para parceiro |
