# Tool: Prospector de Parceiro

## Funcao
Gerar abordagem personalizada para prospectar um parceiro especifico ou segmento.

## Input
Fornecer:
1. **Segmento**: Tipo de negocio (restaurante, academia, barbearia, etc.)
2. **Cidade**: Onde o parceiro esta
3. **Base local**: Quantos membros Megalink na cidade
4. **Nome do estabelecimento** (opcional): Se ja tem um alvo especifico
5. **Canal**: WhatsApp, presencial, telefone

## Output Obrigatorio

```
## Prospeccao: [Segmento] em [Cidade]

### Contexto
- Base Megalink em [cidade]: [X] membros
- Segmento: [tipo de negocio]
- Canal de abordagem: [canal]

### Proposta de Valor (Especifica)
[Por que esse tipo de negocio se beneficia do Clube — max 3 linhas]

| Campo | Sugestao |
|-------|---------|

| Valor | [X]% ou R$[X] |
| Modalidade | [gratuito/pontos/nivel] |
| Estoque | [X ou ilimitado] |
| Validade | [X dias/meses] |

### Script — [Canal]

---

[Mensagem final]
```
