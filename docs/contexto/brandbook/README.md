- **Font principal:** Inter (Google Fonts)
- **Pesos usados:** 400 (regular), 500 (medium), 600 (semibold), 700 (bold), 800 (extrabold)
- **Tamanhos padrao:**
  - Titulo de pagina: 22px / 800
  - Subtitulo: 13-14px / 400
  - Labels: 11px / 700 uppercase
  - Corpo: 13-14px / 400-500
  - KPI grande: 28-36px / 800

## Tom de Voz

### Para o Membro (B2C)
- Acessivel e amigavel
- Usar "voce" (informal)
- Frases curtas, diretas
- Emojis com moderacao (WhatsApp sim, site com cuidado)
- Evitar: jargao tecnico, termos juridicos, linguagem corporativa
- Exemplo bom: "Voce tem 7 giros pra jogar! Gira a roleta e concorre a premios."
- Exemplo ruim: "Prezado cliente, informamos que seu saldo de pontos esta disponivel para utilizacao."

### Para o Parceiro (B2B)
- Profissional mas acessivel
- Tom de parceria (nao de venda)
- Foco no beneficio concreto dele
- Numeros e resultados
- Evitar: "anuncio", "publicidade", "propaganda"
- Exemplo bom: "Seu cupom teve 15 resgates no primeiro mes. Quer criar uma oferta nova?"
- Exemplo ruim: "Informamos que sua campanha publicitaria obteve excelentes resultados."

### Para o Admin (Interno)
- Tecnico e objetivo
- Dados primeiro, opiniao depois
- Linguagem clara sem ser informal demais

## Padroes de Interface

### Area do Membro
- Fundo: `#000b4a`
- Cards: branco com border-radius 20px e box-shadow
- Header: nome + saldo + botao sair
- Botoes: branco com texto `#000b4a` OU `#000b4a` com texto branco

### Dashboard Admin
- Topbar: branca com modulos centralizados
- Sidebar: branca, muda conforme modulo
- Cards de KPI: brancos com borda `#e5e7eb`
- Badges: verde (sucesso), amarelo (pendente), vermelho (erro)

### Painel do Parceiro
- Topbar: branca com nav de paginas
- Cor de destaque: `#6366f1` (roxo/indigo)
- Cards e tabelas: padroes do admin

### Paginas Publicas (indicacao, validacao)
- Fundo: configuravel (default `#000b4a`)
- Card central: branco, border-radius 20px, max-width 480px
- Botao principal: cor configuravel (default `#000b4a`)
